# 第一章、图形推理

图形推理的出题形式：

图形推理的考点：

  1. 位置规律
  2. 样式规律
  3. 属性规律
  4. 数量规律
  5. 空间重构

快速定位某个考点：图形特征

# 1、位置规律

“位置规律” 图形特征：元素组成相同

元素组成相同考虑位置上的变化

考点：

  1. 平移
  2. 翻转、旋转

# 考点 1：平移

  • 方向：直线（上下、左右、写对角线），绕圈（顺 / 逆时针）
  • 步数：恒定、递增 / 减（等差）、等比（没怎么考）

【宫格类题目】（9 宫格、16 宫格）

  • 技巧：看中间，判方向
    1. 绕圈走：中间黑块数量相同，优先考虑内、外圈分开看
    2. 走直线：中间黑块数量不同，优先考虑上下左右
      • 横行黑块数量相同 -- 左右
      • 竖行黑块数量相同 -- 上下

# 考点 2：翻转、旋转

# 翻转

  1. 左右翻转：竖轴对称
  2. 上下翻转：横轴对称

# 选择

  1. 方向：顺 / 逆时针

  2. 常见角度：30°、45°、60°、90°、180° 等

重要思维：

多个元素分开看，对比选项找差异

# 2、样式规律

“样式规律” 图形特征：元素组成相似

考点：

  • 加减同异
  • 黑白运算

# 考点 1：加、减、同、异

  1. 图形相加、相减
  2. 求同（保留相同）
  3. 求异（保留不同）

线条重复出现 -- 加减同异

# 考点 2：“黑白” 运算

  1. 特征：图形轮廓和分割区域相同，内部的颜色不同
  2. 方法：相同位置运算

重难点：

  1. 对比选项找差异
  2. 大块面积一起看

# 3、属性规律

“属性规律” 图形特征：元素组成不相同、不相似

元素组成不同，优先考虑属性

考点：

  • 对称性
  • 曲直性
  • 开闭性

# 考点 1：对称性

  1. 轴对称：” 等腰 “元素出现
  2. 中心对称：平行四边形、N、Z、S 变形图、相同图形正反
  3. 轴对称 + 中心对称：存在相互垂直的对称轴

识别题型 -- 画出对称轴 -- 考虑数量、反向 -- 对称轴与线、点、面的关系

# 对称性的考法

  1. 区别对称轴类型：轴对称、中心对称、轴对称 + 中心对称
  2. 对称轴的数量和方向
  3. 对称轴与图形线、点、面的关系

最” 火 “对称考法 -- 对称轴经过的部分

# 考点 2：曲直性

  1. 全直性
  2. 全曲性
  3. 曲 + 直

# 考点 3：封闭性

  1. 全封闭
  2. 全开放
  3. 半开半闭

# 4、数量规律

” 数量规律 “图形特征：

  • 元素组成不同且属性无规律
  • 数量特征明显

考点：面、线、点、素

# 考点 1：面数量

面是白色的封闭区域

注意：面是白的，黑的是不是面！

考虑数面的情况：

  1. 图形被分割、封闭面明显
  2. 生活化图形、粗线图图形

面的细化考法：

  1. 所有面的形状（三角形、四边形）
  2. 相同面的数量
  3. 最大 / 最小面的形状、属性（对称、曲直）、与外框的关系（相似、相等）

# 考点 2：线数量

  • 直线
  • 曲线

考虑数线的情况：

  1. 直线特征图：多边形、单一直线
  2. 曲线特征图：曲线图形（圆、弧、单一曲线）

线的特殊考点：笔画数（不重复的情况下，能够一笔画成）

笔画特征图：多端点、圆 / 相切 / 相交、日、田、五角星

判断一笔画：

  • 线条之间连通
  • 奇点数量为 0 或 2（奇点：以一个点为中心，发射出奇数条线，注意端点也算）
  • 笔画数 = 奇点数 / 2（奇点数一定为偶数个）

# 考点 3：点数量

线与线的交点为点

注意：切点也算点，端点不算

考虑数点的情况：

点数量特征图：

  1. 线条交叉明显（大树杈）
  2. 多边形或者圆中叉出一些线条
  3. 圆相切或者圆相交较多

点的细软考法：

  1. 切点：相切图较多，切点明显
  2. 框上 / 框内交点：所有图形都有外框，且内部线条与外框存在相交的情况

生活化、粗线条图形常见考法：

  1. 属性（对称、开闭）
  2. 部门数
  3. 面

# 5. 空间重构

折叠方向

排除错误

# 方法 1：相对面

应用：一组相对面同时出现的选项 -- 排除

# 方法 2：相邻面

平面图中构成直角的两条边是同一条边

一列 / 行连着 4 个面，两头的两条边是同一条边

相邻三个面的公共点是唯一的

  • 三个面在一起 -- 优先公共点
  • 三个面没在一起 -- 优先公共边
