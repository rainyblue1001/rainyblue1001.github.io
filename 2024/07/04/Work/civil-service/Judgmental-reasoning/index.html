# 第一章、图形推理

图形推理的出题形式：

图形推理的考点：

  1. 位置规律
  2. 样式规律
  3. 属性规律
  4. 数量规律
  5. 空间重构

快速定位某个考点：图形特征

# 1、位置规律

“位置规律” 图形特征：元素组成相同

元素组成相同考虑位置上的变化

考点：

  1. 平移
  2. 翻转、旋转

# 考点 1：平移

  • 方向：直线（上下、左右、写对角线），绕圈（顺 / 逆时针）
  • 步数：恒定、递增 / 减（等差）、等比（没怎么考）

【宫格类题目】（9 宫格、16 宫格）

  • 技巧：看中间，判方向
    1. 绕圈走：中间黑块数量相同，优先考虑内、外圈分开看
    2. 走直线：中间黑块数量不同，优先考虑上下左右
      • 横行黑块数量相同 -- 左右
      • 竖行黑块数量相同 -- 上下

# 考点 2：翻转、旋转

# 翻转

  1. 左右翻转：竖轴对称
  2. 上下翻转：横轴对称

# 旋转

  1. 方向：顺 / 逆时针

  2. 常见角度：30°、45°、60°、90°、180° 等

重要思维：

多个元素分开看，对比选项找差异

# 思维导图

# 2、样式规律

“样式规律” 图形特征：元素组成相似

考点：

  • 加减同异
  • 黑白运算

# 考点 1：加、减、同、异

  1. 图形相加、相减
  2. 求同（保留相同）
  3. 求异（保留不同）

线条重复出现 -- 加减同异

# 考点 2：“黑白” 运算

  1. 特征：图形轮廓和分割区域相同，内部的颜色不同
  2. 方法：相同位置运算

重难点：

  1. 对比选项找差异
  2. 大块面积一起看

# 思维导图

# 3、属性规律

“属性规律” 图形特征：元素组成不相同、不相似

元素组成不同，优先考虑属性

考点：

  • 对称性
  • 曲直性
  • 开闭性

# 考点 1：对称性

  1. 轴对称：” 等腰 “元素出现
  2. 中心对称：平行四边形、N、Z、S 变形图、相同图形正反
  3. 轴对称 + 中心对称：存在相互垂直的对称轴

识别题型 -- 画出对称轴 -- 考虑数量、反向 -- 对称轴与线、点、面的关系

# 对称性的考法

  1. 区别对称轴类型：轴对称、中心对称、轴对称 + 中心对称
  2. 对称轴的数量和方向
  3. 对称轴与图形线、点、面的关系

最” 火 “对称考法 -- 对称轴经过的部分

# 考点 2：曲直性

  1. 全直性
  2. 全曲性
  3. 曲 + 直

# 考点 3：封闭性

  1. 全封闭
  2. 全开放
  3. 半开半闭

# 思维导图

# 4、数量规律

” 数量规律 “图形特征：

  • 元素组成不同且属性无规律
  • 数量特征明显

考点：面、线、点、素

# 考点 1：面数量

面是白色的封闭区域

注意：面是白的，黑的是不是面！

考虑数面的情况：

  1. 图形被分割、封闭面明显
  2. 生活化图形、粗线图图形

面的细化考法：

  1. 所有面的形状（三角形、四边形）
  2. 相同面的数量
  3. 最大 / 最小面的形状、属性（对称、曲直）、与外框的关系（相似、相等）

# 考点 2：线数量

  • 直线
  • 曲线

考虑数线的情况：

  1. 直线特征图：多边形、单一直线
  2. 曲线特征图：曲线图形（圆、弧、单一曲线）

线的特殊考点：笔画数（不重复的情况下，能够一笔画成）

笔画特征图：多端点、圆 / 相切 / 相交、日、田、五角星

判断一笔画：

  • 线条之间连通
  • 奇点数量为 0 或 2（奇点：以一个点为中心，发射出奇数条线，注意端点也算）
  • 笔画数 = 奇点数 / 2（奇点数一定为偶数个）

# 考点 3：点数量

线与线的交点为点

注意：切点也算点，端点不算

考虑数点的情况：

点数量特征图：

  1. 线条交叉明显（大树杈）
  2. 多边形或者圆中叉出一些线条
  3. 圆相切或者圆相交较多

点的细软考法：

  1. 切点：相切图较多，切点明显
  2. 框上 / 框内交点：所有图形都有外框，且内部线条与外框存在相交的情况

# 5. 空间重构

折叠方向

排除错误

# 方法 1：相对面

应用：一组相对面同时出现的选项 -- 排除

# 方法 2：相邻面

平面图中构成直角的两条边是同一条边

一列 / 行连着 4 个面，两头的两条边是同一条边

相邻三个面的公共点是唯一的

  • 三个面在一起 -- 优先公共点

  • 三个面没在一起 -- 优先公共边

  1. 属性（对称、开闭）
  2. 部门数
  3. 面

# 第二章、类比推理

# 1、语义关系

# 考点 1：近义关系、反义关系

1. 近义关系：词语意思相近  例：谦虚∶虚心     走投无路∶山穷水尽
1. 反义关系：词语意思相反  例：慎重∶草率     高瞻远瞩∶鼠目寸光

# 考点 2：比喻义、象征义

1. 比喻义：用跟甲事物有某种相似之处的乙事物来说明甲事物  例：月亮∶小船
1. 象征义：借助某物的具体形象，以表现某种抽象的概念、思想和情感  例：鸽子∶和平

# 2、逻辑关系

# 考点 1：全同关系

全同关系：两个词语意思完全相同 例：蝉∶知了

# 考点 2：并列关系

  1. 矛盾关系：是指只有对立的两种情况，不存在第三种情况，二者是非此即彼的 关系 例：动∶静  等腰∶非等腰
  2. 反对关系：是指在所列的两种情况之外，还存在其他情况，二者不是非此即彼 的关系  例：水稻∶小麦  草鱼∶鲤鱼

# 考点 3：包容关系

1. 种属关系：A 是 B 的一种  例：橡皮∶文具
1. 组成关系：A 是 B 的一个组成部分  例：氧气∶空气
1. 区分：具备包容关系的词语中，若能用 “是” 造句子且造得通顺，则词语间为种 属关系

# 考点 4：交叉关系

判定技巧 —— 造句子：

  • 有的 A 是 B 且  有的 A 不是 B
  • 有的 B 是 A 且 有的 B 不是 A
  • 例：男教师∶公办教师

# 考点 5：对应关系

高频对应关系：

  1. 材料 例：玻璃：鱼缸
  2. 工艺 例：酱油：发酵
  3. 功能 例：药物：治病
  4. 属性 例：金属：导电性
  5. 时间顺序 例：设计：制作：售卖
  6. 因果 例：营养不良：发育迟缓
  7. 方式目的 例：扶贫：脱贫

# 3. 语法关系

通常在运用语义关系或逻辑关系找不到答案的情况下，再考虑语法关系

1. 主谓关系  例：记者∶采访 裁缝∶剪裁
2. 动宾关系  例：修剪∶枝叶 排除∶嫌疑
3. 主宾关系  例：农民∶水稻 医生∶患者

# 第三章、定义判断

# 1、读得准

# 考点 1：主体、客体

关注主体、客体，主体、客体错误的选项一定不符合定义

# 考点 2：句式

常见句式引导词：

大前提：当…… 时、在…… 过程中、在…… 后

方式：通过、利用

目的：以、达到

原因：因为、由于

结果：导致、从而

# 2、都得快

# 考点 1：长定义

  1. 优先看定义所在的句子
  2. 有解释说明，重点关注解释说明
  3. 常见的解释说明引导词：即、包括、也就是说、以下情形……

# 考点 2：多定义

  1. 问哪个定义优先看哪个定义
  2. 在两个选项中纠结时，再与其他定义做对比

# 考点 3：拆词

运用：定义本身词汇通俗易懂，优先关注；题干读不懂或选项纠结时，可成为重 要破题点

# 第四章、逻辑判断（翻译推理）

# 1、翻译推理

题型特征：

  1. 题干出现明显的逻辑关联词

  2. 提问方式为 “可以推出”“不能推出”

    解题思维： ①先翻译 ② 再推理

# 考点 1：“前推后”

典型逻辑关联词：

  • 如果…… 那么……
  • 若…… 则……
  • 只要…… 就……
  • 所有…… 都……
  • 为了…… 一定……

速记口诀：如果、就、都、一定

# 考点 2：“逆否等价”

符号表示：A→B = －B→－A

文字表示：肯前必肯后，否后必否前，否前肯后不必然（可能 / 可能不）

# 考点 3：“后推前”

  • 典型逻辑关联词：
  • 只有…… 才……
  • 不…… 不……
  • 除非…… 否则不……
  • …… 是…… 的基础 / 假设 / 前提 / 关键
  • …… 是…… 的必要条件 / 必不可少的条件

# 考点 4：“且” 与 “或”

二者同时成立

  • 符号表示：A 且 B
  • 典型逻辑关联词：
    • …… 和……
    • …… 并且……
    • 既…… 又……
    • 不仅…… 而且……
    • 虽然…… 但是……

二者至少一个成立

  • 符号表示：A 或 B
  • 典型逻辑关联词：
    • …… 或者……
    • 或者…… 或者……
    • …… 和…… 至少有一个

# 考点 5：“否一推一”

符号表示：A 或 B 为真，-A→B/-B→A

文字表示：当 “或” 关系为真时，否定一项可以得到另一项

# 考点 6：“德・摩根定律”

-（A 且 B）＝-A 或 - B-

（A 或 B）＝-A 且 - B

# 思维导图

# 2、组合排列

# 考点 1：排除法与代入法

  1. 排除法

  2. 代入法

  • （1）题干条件有真有假

  • （2）提问方式为 “补充以下哪个条件”

  • （3）提问方式为 “可能”“不可能”

题干信息确定，优先使用排除法；题干信息不确定，优先使用代入法

# 考点 2：推理起点

  1. 确定信息：以题干中给出的确定信息为推理起点
  2. 最大信息：以题干中出现次数最多的信息为推理起点
  3. 假设法：最大信息不确定，或代入复杂

# 考点 3：材料题

  1. 与非材料题的解题方法一致
  2. 通过材料直接推出来的结论可以直接应用于所有题目

# 思维导图

# 3、日常结论

题型特征：题干为类言语的表述，无明显逻辑关联词；提问方式为 “可以推 出”“不能推出” 等

解题思维：

  1. 不选

    • （1）逻辑错误
    • （2）无中生有
    • （3）偷换概念

  2. 慎选

    • （1）比较性的词：比……、越来越……、更等

    • （2）绝对化的词：一定、必须、肯定、只要…… 就……、只有…… 才…… 等

    • （3）程度性的词：最 / 极大 / 很等

  3. 优选

    • 可能性表述：可能、有的、有些等

# 4、逻辑判断

论证的基础知识：

1. 论点：观点、态度、结论
2. 论据：证明论点成立的原因、例子
3. 论证：论点与论据之间的联系

找论点：

  1. 结合提问方式
  2. 关键词：因此、所以、认为、由此推出、据此可知等
  3. 首尾句原则

找论据：

  1. 关键词：由于、因为、理由是、根据等
  2. 就近原则：如果文段有论据，一般紧挨着论点

解题步骤： 第一步：看清问法（削弱 / 反驳 / 质疑，加强 / 支持、前提 / 假设）

# 考点 1：削弱之否定论点

题干特征： （1）只有论点，无论据 （2）论点、论据话题一致

选项特征：与论点表述的意思相反

# 考点 2：削弱之拆桥

题干特征： （1）论点、论据话题不一致 （2）提问方式为 “削弱论证” 时，可优先考虑拆桥

选项特征：指出论点与论据之间没有必然联系

# 考点 3：削弱之否定论据

题目特征：

（1）题干中出现支持方、反对方观点相反的情况，经常采用否定论据来削弱 （2）题干存在论据且找不到否定论点和拆桥的选项时，考虑否定论据

选项特征：与论据表述的意思相反

# 考点 4：削弱之因果倒置和他因削弱

题干特征： 论点具有因果关系

选项特征：

（1）因果倒置：将论点中的因果关系顺序颠倒

（2）他因削弱：在论点原因 1 的基础上，选项增加另一个同时存在的原因 2，也 可能导致该结果，削弱的是原来原因 1 的重要性或者唯一性

# 特殊提问：不能削弱

# 削弱论证的思维导图

# 考点 5：加强之补充论据

题干特征： （1）只有论点，无论据 （2）论点、论据话题一致

选项特征：

解释原因 —— 说明论点成立的原因

举例子 —— 证明论点成立的例子

# 考点 6：加强之搭桥

题干特征：

（1）提问方式为 “前提”“假设” 时，优先考虑搭桥

（2）论点、论据话题不一致 选项特征：指出论点与论据之间有关系

# 考点 7：加强之必要条件

题目特征：提问方式为 “前提”“假设”，且无搭桥项；或只有论点

选项特征：选项为论点成立的必要条件（没它不行）
