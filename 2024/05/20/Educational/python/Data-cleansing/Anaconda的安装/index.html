任务： 安装 Anaconda
如何安装 Anaconda？Anaconda 这是一个用于科学计算和数据分析的 Python 发行版。Anaconda 包含了许多常用的库和工具，使得在各种操作系统上轻松地设置和管理 Python 环境，在本地计算机进行 Anaconda 的安装。

# 任务： 安装 Anaconda

# 1、Anaconda 的下载（①国内镜像网站、②Anaconda 官网）

More info: ①清华大学开源镜像网站
选择电脑选择对应版本（自己下载的版本：Anaconda3-5.3.1-Windows-x86_64.exe）

下载完成如下：

More info: ②Anaconda 官网
进入官网，点击下载图标，官网会根据电脑对应的版本进行下载

下载完成如下：

# 2、 Anaconda 的安装：

①打开刚才下载好的文件（自己选择的是官网下载的 Anaconda3-2023.09-0-Windows-x86_64.exe）出现以下界面，选择 Next

②出现协议告知界面，选择 “I agree”

③出现新界面，选择 “All Users”, 然后点击 “next”

④出现安装目录（自己选择放在 D:\Python\Anaconda3），然后点击 “next”

⑤出现新界面，全部打勾选择（1 创建快捷方式，2 将 Anaconda 注册为系统 Python3.11，3 完成后清除包缓存），然后点击 “Install”

⑥出现安装进度条界面，等待直到出现 “completed”，然后点击 “next”- 继续点击 “next”

⑦完成安装

# 3、配置环境变量

①点击 win，进入搜索 “环境”，点击 “编辑系统环境变量 “（也可从 “此电脑” 属性进入 “系统环境变量”

②出现新界面，点击 “环境变量（N）“

③选择 “环境变量（S），找到 “Path”，点击编辑，新建以下 5 条目录（根据自己实际安装目录进行改动）

点击确定，保存退出

# 4、测试配置是否成功

①Win+r，输入 cmd，进入命令提示符
输入 “python”，检查是否有 python 环境

②输入 conda info 查询 conda 信息 ，如图为 conda 环境配置成功

⑤win 搜索 Anaconda，点击打开 “Anaconda Navigator”

出现界面如下，即全部完成

到此，完成 Anaconda 安装
